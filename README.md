# 千年

一个致敬经典武侠网游《千年》的游戏项目，基于 [Cocos Creator](https://www.cocos.com/creator) 开发，目标平台为 Web 与 iOS / Android 原生 App。

> 武林千年，江湖路远。

## 仓库结构

```
qiannian/
├── client/      # Cocos Creator 3.8 客户端
│   └── assets/
│       ├── scenes/            # 场景
│       ├── scripts/
│       │   ├── core/          # 引擎无关逻辑（禁止 import 'cc'，可复用到服务端）
│       │   ├── game/          # 游戏组件
│       │   └── ui/            # UI 组件
│       └── resources/data/    # 配置表 JSON
└── docs/        # 开发文档（环境搭建见 docs/SETUP.md）
```

## 当前进度：角色控制演示

第一个可运行版本，聚焦角色控制手感，暂不包含地图、UI 与 NPC：

- **八方向身位**：角色按 45° 一档共 8 个朝向，转向沿最短弧逐步完成，亦可原地转向
- **四种状态**：站立（呼吸起伏）、走路、奔跑（前倾、步频加快）、打坐（盘膝入定，灵气环绕）
- 角色形象与参照网格均为程序化矢量绘制，后续替换美术资源时只需重写渲染组件
- 相机跟随（世界反向移动），状态/朝向实时显示

### 操作

| 按键 | 功能 |
| --- | --- |
| `W` `A` `S` `D` / 方向键 | 八方向移动 |
| `Shift`（按住） | 奔跑 |
| `R` | 切换 跑 / 走 模式 |
| `X` | 打坐 / 起身（移动自动起身） |
| `Q` / `E` | 原地转向 |

### 运行

1. 按 [docs/SETUP.md](docs/SETUP.md) 安装 Cocos Creator 3.8.x 并打开 `client/` 项目
2. 打开场景 `assets/scenes/main.scene`，点击预览即可
