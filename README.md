# 千年 · 网页版演示

一个致敬经典武侠网游《千年》的网页单机演示，使用原生 HTML5 Canvas + JavaScript 实现，无任何框架和构建工具，打开即玩。

> 武林千年，江湖路远。

## 快速开始

**方式一：直接打开**

双击 `index.html` 即可在浏览器中运行。

**方式二：本地服务器（推荐）**

```bash
python3 -m http.server 8765
# 然后访问 http://localhost:8765
```

推荐使用 Chrome / Edge / Safari 等现代浏览器。

## 操作说明

| 操作 | 说明 |
| --- | --- |
| 鼠标左键点击地面 | 移动（自动寻路） |
| 点击怪物 | 自动追击并攻击 |
| 点击 NPC | 对话 / 买卖 / 接任务 |
| 点击地上的物品 | 拾取 |
| `W` `A` `S` `D` / 方向键 | 走动 |
| `1` ~ `4` | 快捷栏（药品 / 武功） |
| `C` | 人物窗口 |
| `B` | 背包窗口 |
| `V` | 武功窗口 |
| `Q` | 任务窗口 |
| `H` | 操作说明 |
| `ESC` | 关闭窗口 / 系统菜单 |

## 游戏内容

### 世界

- 2.5D 斜 45° 等距视角，所有画面元素均为 Canvas 程序化绘制
- 两张地图：**玄菟村**（安全区）与**城外荒野**，通过传送点互通
- 十二时辰昼夜循环，入夜后画面渐暗
- 小地图、伤害飘字、刀光 / 治疗 / 升级特效、程序化音效

### 武功系统（千年特色）

武功不靠等级解锁，而是**用进废退**：

- **拳法 / 剑术 / 刀术**：空手出招练拳法，持剑练剑术，持刀练刀术，按出手次数提升熟练度，从一重直至十二重圆满，每升一重攻击力大增
- **内功**：通过消耗内力（施展治愈术、轻功）修炼，提升防御与内力上限
- 主动武功：
  - **治愈术**：运转内息疗伤，效果随内功重数增强
  - **轻功**：移动速度大幅提升，持续消耗内力

### 战斗

- 点击怪物自动追击攻击，有会心一击（暴击）
- 主动怪有仇恨范围，会追击玩家，脱战后回血归位
- 怪物死亡后定时刷新；荒野东北的山贼营寨有 Boss **山贼头目**
- 死亡损失少量经验，被送回玄菟村

### 怪物

| 怪物 | 习性 | 掉落 |
| --- | --- | --- |
| 野鹿 | 被动 | 鹿茸 |
| 野狼 | 主动 | 狼皮、金创药 |
| 蟒蛇 | 主动 | 蛇胆、回气丹 |
| 山贼 | 主动 | 大金创药、柴刀、布衣 |
| 山贼头目（Boss） | 主动 | 青龙刀、铁甲 |

### NPC 与任务

- **村长**：任务链——【狼患】（猎杀 5 头野狼）→【剿灭山贼】（讨伐山贼头目）
- **王铁匠**：武器防具店（木剑、铁剑、柴刀、青龙刀、铁拳套、布衣、皮甲、铁甲）
- **李大夫**：药铺（金创药、大金创药、回气丹），可免费疗伤

### 物品与存档

- 16 种物品：武器 / 防具 / 药品 / 材料，支持装备、使用、买卖、掉落拾取
- 进度自动保存到浏览器 localStorage（每 30 秒及关闭页面时），标题画面可继续游戏

## 项目结构

```
qiannian/
├── index.html          # 页面骨架与 UI 布局
├── css/
│   └── style.css       # 全部界面样式（古风暗金主题）
└── js/
    ├── data.js         # 游戏数据：物品 / 怪物 / 武功 / 任务 / NPC
    ├── world.js        # 地图生成（种子随机）、A* 寻路
    ├── entities.js     # 实体：玩家 / 怪物 AI / NPC / 掉落物
    ├── render.js       # 等距视角渲染、程序化绘图、特效、小地图
    ├── ui.js           # 窗口 / HUD / 商店 / 对话框 / 音效
    └── game.js         # 主循环、输入、战斗、任务、存档
```

脚本按上述顺序以普通 `<script>` 标签加载（非 ES Module），因此 `file://` 协议下也能直接运行。

## 当前限制

- 单机演示，暂无联网功能
- 标签页切到后台时游戏暂停（`requestAnimationFrame` 节流），回到前台自动继续
